# Chat Room - FSE project
### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Start Server Side
```
node server/index.js
```

### Start Mongodb
```
mongod
```
